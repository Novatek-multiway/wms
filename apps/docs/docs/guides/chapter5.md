## 5.1 大屏界面

在浏览器中输入系统大屏的地址后，进入大屏界面。  
大屏界面的整体布局：

- 顶部显示仓库名或项目名称，名称左侧显示今日完成任务数，右侧显示累计完成任务数。
- 中间位置，显示仓库3D立体图，可以通过右侧工具栏按钮，切换是否展示空库位、开启/禁止3D旋转已经隐藏底部相关统计面板。
- 底部从左往右，分别展示了按物料实时库存统计数据、库位空满统计、未完成任务信息展示、空满托盘库存统计数据以及当天出入库累计数据。
